# Library-Management-System

This is a Library management app project that the user can sign up and sign in and rent from a catalogue of books.

## Pages

-   Home Page
-   Login and Signup pages
-   Student portal
-   Request Book 
-   Available Books
-   Return Books
-   Admin portal
-   Add Books Form
-   Add Student
-   Delete Student 
-   Issue Books

## Partials
-   Navbar
-   SideBar
-   Footer
